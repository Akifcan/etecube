# Run Docker

`docker run --name etecube-db  -e POSTGRES_PASSWORD=etecube -e POSTGRES_DB=etecube -d -p 5436:5432 postgres`

# In Backend Folder

 - `npm install`
 - `npm run dev`

# In Frontend Folder

 - `npm install`
 - `npm run dev`
